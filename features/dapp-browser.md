# DApp Browser

> All of Web3 — right from the wallet. No extensions, no switching.

---

## What Is It?

The DApp Browser is a built-in web browser that lets you use **decentralized applications** directly from QR Wallet. Your wallet connects automatically — like MetaMask, but without extensions.

## How to Use

1. Open the **Discovery** section
2. Select a DApp from the catalog or enter a **URL manually**
3. Wallet automatically connects to the site
4. When the DApp requests a transaction — a confirmation modal appears
5. Review details and confirm with biometrics

---

## Features

- **Web3 Provider Injection** — automatic wallet connection
- **Transaction signing** — approve actions through the wallet
- **WalletConnect** — support for WalletConnect protocol
- **Favorites** — save your favorite DApps
- **Recent** — history of visited DApps
- **Navigation** — back, forward, refresh, home

---

## Security

- Every transaction requires **explicit confirmation** via biometrics
- **Transaction details** displayed before signing
- Malicious sites and store redirects blocked
- DApp **has no access** to your private key
