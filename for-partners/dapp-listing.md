# DApp Listing

> Feature your DApp in the QR Wallet Discovery catalog.

---

## What Is Discovery?

Discovery is the built-in DApp catalog in QR Wallet. Users can open your DApp directly from the wallet with automatic connection.

---

## Categories

| Category | Description |
|----------|------------|
| **DeFi** | DEXes, lending, yield farming |
| **NFT** | Marketplaces, collections |
| **Gaming** | Web3 games |
| **Bridge** | Cross-chain bridges |
| **Social** | Social protocols |
| **Tools** | Utilities, analytics |
| **Airdrops** | Current airdrops |

---

## Requirements

To be listed, your DApp must:

- Have a working web interface (mobile responsive)
- Support standard wallet connection (EIP-1193 / WalletConnect)
- Operate on at least one supported network
- Contain no malicious code

---

## What You Get

- **Catalog card** — name, icon, description, category
- **Featured position** — for strategic partners
- **Seamless UX** — wallet connects automatically
- **Partnership quests** — ability to create user quests

---

## How to Apply

Submit a [partnership request](partnership-request.md) with:

1. DApp name and description
2. URL
3. Category
4. Supported networks
5. Logo (SVG or PNG, 256×256)
