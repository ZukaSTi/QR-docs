# Partnership Request

> Become part of the QR Wallet ecosystem.

---

## Partnership Types

| Type | Description |
|------|------------|
| **Payment Integration** | Accept crypto payments via QR codes |
| **DApp Listing** | Feature in the Discovery catalog |
| **Partnership Quests** | Quests for QR Wallet users |
| **Token Integration** | Add your token to the registry |
| **Strategic Partnership** | Joint campaigns and integrations |

---

## How to Apply

### Email

Write to **partners@qr.xyz** with:

1. **About you** — project name, website, description
2. **Partnership type** — what you want to integrate
3. **Details** — networks, tokens, scale
4. **Contact** — how to reach you

### In-App

In Discovery → **Partner Request** — built-in application form.

---

## What's Next?

1. **Review** — we'll review your application (1–3 business days)
2. **Discussion** — we'll reach out to discuss details
3. **Integration** — API keys and documentation provided
4. **Testing** — joint testing
5. **Launch** — integration goes live
